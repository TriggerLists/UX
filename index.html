UX Trigger List

Before you set off on implementing a UX flow or a new UI, read over this list to see if you have forgotten or broken an maxims or best practices.

  • Don't make me think!
  • Can anything be removed?
  • Are expectations set appropriately?
  • Is copy clear and only possibly interpreted one way?
  • What assumptions have been made about the user's knowledge?
  • Are the user's current emotions being spoken to?
  • Is the user wondering if anything is happening in the background?
